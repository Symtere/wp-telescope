## 👽🤖 Installation
❗😱 vhost :  `telescope.local`

```php
cd www/
git clone git@github.com:Symtere/wp-telescope.git telescope
cd telescope
npm i
```

Déplacer le fichier `config/install.sh` à la racine du projet

```php
sh install.sh
```
